\doxysubsection{Detailed Description}
Base class for a game. 

This class creates a Base for creating and starting the Game.

Usage\+:
\begin{DoxyItemize}
\item Inherit from \doxylink{classte_1_1GameBase}{Game\+Base}
\item Override \doxylink{classte_1_1GameBase_a0a8905a011a8ab2cd4b44ec611ff2aeb}{init()} and call Parent init to load game objects and logic
\item Call \doxylink{classte_1_1GameBase_accb4b63d1277e46131d66a66c0e1e63b}{run()} to start the game loop 
\end{DoxyItemize}
